# Neosubhamoy Tap

## How do I install these formulae?

`brew install neosubhamoy/tap/<formula>`

Or `brew tap neosubhamoy/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "neosubhamoy/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
